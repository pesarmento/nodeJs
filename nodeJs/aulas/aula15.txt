Estruturas de controle
Laços de repetições
laço for (número determinado de repetições)

- É a única estrutura que permite criar internamento uma
variavel do tipo númerico inteiro 
- Início ; condição ; incremento/decremento
- O laço é repetido enquanto a condição é verdadeira

for(let i = 1; i < 10; i++){

}


Criando um laço FOR dentro de outro laço FOR :
for() {
    for() {

    }
}

obs:
1) Usar nomes diferentes de variaveis internas 
2) O incremento/decremento do laço só é executado após a finalização do laço for interno
